Intro

Welcome to my portfolio! Here you can find my best artwork and well developed webpages.

I enrolled in Eaton RESA Career Prep Center during the 2017-2018 school year, my senior year in high school. In this class I've learned several functions of Photoshop, Dreamweaver, and Adobe Animate. I struggle creating well developed images from my imagination with my hand, so I've included many images I've edited. I excelled at web design during my experience in this class. Not only did I enjoy it more but my work reflects this passion with the content and functions it contains. Take a look around and enjoy yourself!

Work

This piece of work was made while learning how to illustrate depth in a photo. This project taught me to shape an image using the perspective tool in Photoshop and how to place shadows in correspondance to a ligt source.

I did not create this from scratch but the photo is highly rendered with several changes in hue, saturation, and contrast. Though I didn't intend to create this kind of work before I started working on it, I realized this is an aesthetic I really enjoy creating and viewing. I learned how to mix and combine color variation to create a specific outcome.

Another piece of work I created using filters and alterations of an image I found online (with free use to modify). This image, and the others like it, are what I retained most from photoshop: the alteration of images. I enjoy this type of work opposed to freehand drawing.

This image was created with symmetry in mind and ended up here for the precision it has. This fighting style intrigues me, so this holds sentimental value as well as artistic value. I learned how to crop, align, and rotate images to create a radial symmetry design.

This webmap was made with time and patience as each box took time to line up precisely. The project required us to place a box over each part of a website's design. I appreciate this work for the time it took and how well the colors worked together to create a theme.

For this piece we created an african textile based on three different colors to create a central theme. The design is pleasing and the colors all work together to portray "warmth", a characteristic resembling Africa. This taught me how to use colors together for a theme.

This love story is by far my favorite piece that I've created from scratch. The visual portrayed represents two things: a love story on a physical level and a slightly perverted sense of humor. I'm proud of how well it turned out for my level of drawing skill and felt it deserved to be shown off. While creating this I learned how to represent a feeling through a drawing.

This creation was inspired by a travel poster design we had to follow. I had to create one based around Jupiter and its barren-described setting. I included this drawing for the depth it represents and the way the over arching theme is evident. I enjoy looking at this because it makes me think I can jump from one "balloon" to the next, which plays a short movie in my mind.

Finally, this wordmark is included because it's another aesthetic I enjoy all too much. The hazy grime surrounding the edges of my name gives the word a distinct personality. I learned how to edit the individual edges of a text while creating this.

About

The Digital Media and Design program gave me the opportunity to build my artistic ability and introduce me to web design. Since the beginning of the year I've learned how to use most tools in Photoshop, create logos, webmaps, wordmarks, unique images, and cropped photos to make them look as if they weren't edited at all. I also learned how to make gifs, create short animations, create websites from scratch using HTML and CSS, how to design a website, and how to upload them. This portfolio highlights my best works from the categories I've just discussed.
